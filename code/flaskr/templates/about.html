About the Project

This application was developed as a Hackbright Academy four-week project. It uses publicly available data from DataSF to predict the ambulance arrival time at a given address and calculate some ambulance system statistics.

The prediction is based on a random forest regression model fitted to data recorded between 2000 and 2019. The graph below shows the importance of the various features used in the model.

The uncertainty on the predicted ambulance arrival time is approximately 3.5 minutes. Additional features, such as live travel time to the incident, ambulance post locations, commuter-adjusted population estimates, and homelessness data would likely improve the accuracy of the prediction.

To access the code, visit the project's GitHub repo.
